# File and Directory Management


### Rename file - directory
use mv command for rename
```bash
$ mv .old-file .new-file   
```

### Remove file - directory (recursivly)
-r = recursivly

-f = force
```bash
$ rm -r .node_modules  
```

### Tree view :)
-s = folder and file size (MB - KB)

-h = size mode humanfriendly

```bash
$ tree -s -h 
```
